# BBS Shojo游戏 - 实施计划

## 整体目标
基于细化的游戏设计文档，按优先级实现所有功能，确保每个模块都符合复古BBS风格和记忆碎片机制的设计需求。

---

### 步骤1：项目初始化与环境搭建
**指令**：
1. 创建项目文件夹结构：
   - 主文件夹：bbs-shojo-game
   - 子文件夹：src/（存放代码）、assets/（存放资源）、docs/（存放文档）
2. 安装必要的Python库：`pip install pygame mido ascii-art`

**测试**：
- 运行以下代码，验证Pygame是否正常运行：
  ```python
  import pygame
  pygame.init()
  screen = pygame.display.set_mode((800, 600))
  pygame.display.set_caption("BBS Shojo Game Test")
  running = True
  while running:
      for event in pygame.event.get():
          if event.type == pygame.QUIT:
              running = False
  pygame.quit()
  ```
- 如果能打开一个800x600的窗口，说明环境配置成功。

### 步骤2：实现像素精度渐变系统（高优先级1）
**指令**：
1. 设计一个函数，接受图像路径和记忆碎片数量作为参数
2. 根据记忆碎片数量，动态调整图像的缩放比例和模糊程度：
   - 记忆碎片0：最大缩放（像素化），最大模糊
   - 记忆碎片100：原始图像大小，无模糊
3. 实现平滑过渡效果，避免跳跃感
4. 添加测试代码，验证不同记忆碎片数量下的图像效果

**测试**：
- 加载一张测试图像，逐步增加记忆碎片数量，观察图像是否从像素化逐渐变为清晰
- 检查过渡效果是否平滑，没有明显的卡顿

---

### 步骤3：实现CRT老化模拟效果（高优先级2）
**指令**：
1. 设计一个函数，接受游戏画面表面作为参数
2. 实现以下效果：
   - 扫描线：每隔一行绘制半透明的黑色线条
   - 随机噪点：在画面上添加随机的黑白像素
   - 偏色：轻微调整RGB通道的值，模拟CRT显示器的偏色
   - 烧屏效果：叠加轻微的图像残留痕迹
3. 将效果叠加在游戏画面之上，不影响原始交互

**测试**：
- 运行游戏，观察画面是否有CRT显示器的复古效果
- 检查效果是否不影响游戏的交互和性能
