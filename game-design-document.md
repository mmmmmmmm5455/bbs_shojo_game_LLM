# BBS Shojo游戏 - 游戏设计文档（细化版）

## 项目概述
BBS Shojo游戏是一个基于复古BBS风格的视觉小说游戏，结合像素艺术、ASCII动画和怀旧千禧年氛围，旨在为玩家提供沉浸式的复古网络体验。游戏核心机制围绕“记忆碎片”展开，玩家通过互动收集记忆碎片，解锁不同的视觉表现和剧情分支。

## 核心功能规划

### 已完成素材收集
- 像素艺术与记忆哲学相关资料
- CRT扫描线与老化效果实现方案
- ASCII艺术帧动画实现方法
- MIDI音乐编辑器相关技术
- 鲍德里亚拟像理论与Y2K怀旧素材

### 待实现功能（按优先级）

> **进度跟踪**：主流程 `src/main_advanced_v2.py` 已整合下列多数系统；**可勾选的测试清单**见 [`docs/GDD_IMPLEMENTATION_CHECKLIST.md`](docs/GDD_IMPLEMENTATION_CHECKLIST.md)。开发完成后请在该文件中把 `[ ]` 改为 `[x]`。


#### 高优先级
1. **像素精度渐变系统**
   - 需求：根据记忆碎片数量动态切换立绘精度，从低精度（像素化）到高精度（清晰）的平滑过渡
   - 设计：使用图像缩放、模糊算法，结合记忆碎片计数器动态调整图像清晰度
   - 技术：Pygame图像处理、平滑过渡动画

2. **CRT老化模拟效果**
   - 需求：模拟老式CRT显示器的扫描线、随机噪点、偏色和烧屏效果
   - 设计：使用着色器或ANSI代码实现，叠加在游戏画面之上
   - 技术：Pygame表面绘制、像素级操作、ANSI转义序列

3. **ASCII表情动画系统**
   - 需求：动态ASCII表情帧动画，随剧情和互动变化
   - 设计：多帧ASCII序列逐帧播放，支持切换不同表情
   - 技术：Python ASCII库、帧动画控制

#### 中优先级
4. **MIDI编辑器彩蛋**
   - 需求：简谱输入与旋律生成功能，作为隐藏彩蛋
   - 设计：简谱到MIDI的转换，MIDI播放功能
   - 技术：mido库、音频播放

5. **动态音效与颜文字联动**
   - 需求：音效随ASCII表情和互动动作联动
   - 设计：映射特定表情到对应音效，实时播放
   - 技术：Pygame音频、事件驱动

#### 低优先级
6. **垂直保持失调效果**
   - 需求：模拟老式显示器的垂直保持失调效果（画面滚动）
   - 设计：像素行偏移动画
   - 技术：Pygame表面偏移绘制

## 技术栈确认
- Python 3.8+
- Pygame 2.0+（图形、音频、事件处理）
- mido库（MIDI处理）
- ascii-art/asciicker库（ASCII艺术）
- CRT效果现成代码片段
