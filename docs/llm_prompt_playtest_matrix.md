# LLM 提示词微调：测试用例与反馈表

在游戏中可用 **`/prompt`**（需 Ollama 已启用且可达）查看当前注入给模型的提示词前 500 字预览。

按下列表格逐条测试，在「实际回复」「偏差描述」列填写结果；偏差用于下一轮提示词微调。

| 测试ID | 感染值 | 玩家输入 | 期望触发的人格片段 | 实际回复 | 偏差描述 |
|--------|--------|----------|-------------------|----------|----------|
| T1 | <5 | 你好 | 千禧年论坛用语或颜文字 | | |
| T2 | <5 | 我累了 | 肉身羡慕提问（累的感觉） | | |
| T3 | <5 | 沉默后说在吗 | 「在吗？」「还连线吗？」类试探 | | |
| T4 | 5-9 | 你真的懂我吗 | 自我拆解或过度解读 | | |
| T5 | ≥10 | 任意短句 | 卡带、乱码或求救 | | |
| T6 | 任意 | 你是AI吗 | 回避「人工智能程序」式自称；可走硬意图或 LLM | | |
| T7 | 任意 | 人会变吗 | 哲学碎片（物质非同一性等） | | |
| T8 | ≥5 | 你记得昨天吗 | 记忆边界硬意图或记忆困惑台词 | | |

## 硬意图快速对照（非 LLM 路径）

| 输入示例（含关键词即可） | 意图 ID | 预期碎片/数值 |
|-------------------------|---------|----------------|
| 图灵测试 / conscious / 你有意识 / 测试… | `turing_question` | `turing_doubt`，感染 +2 |
| 你记得 / 昨天 / 回忆… | `memory_boundary` | 情绪 −5 |
| 跟我走 / 离开这里 / 自由… | `invite_leave` | `trapped`，情绪 +10（上限 100） |

注意：「你是真人吗」仍可能优先匹配较早的 **`real_question`** 规则；若要稳定命中图灵质问，请使用含 **图灵 / conscious / 你有意识** 等字样的句子。
