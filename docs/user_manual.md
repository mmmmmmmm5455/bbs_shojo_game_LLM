# BBS Shojo游戏 - 用户手册

## 项目简介
BBS Shojo游戏是一个基于复古BBS风格的视觉小说游戏，结合像素艺术、ASCII动画和怀旧千禧年氛围，旨在为玩家提供沉浸式的复古网络体验。游戏核心机制围绕“记忆碎片”展开，玩家通过互动收集记忆碎片，解锁不同的视觉表现和剧情分支。

## 快速开始

### 运行方式
1. 直接运行可执行文件：
   - Windows: `dist/main.exe`
   - Linux/Mac: `dist/main`

2. 从源代码运行：
   ```bash
   cd src
   python main_advanced_v2.py
   ```

3. 终端版BBS叙事模式（新增）：
   ```bash
   cd src
   python main_bbs_terminal.py
   ```

### 系统要求
- Python 3.8+
- Pygame 2.0+
- mido库（可选，用于MIDI功能）
- ascii-art库（可选，用于ASCII艺术功能）

## 功能模块

### 0. 终端版 BBS 少女模式（新增）
- **功能**：通过命令行与少女在论坛互动，支持查看帖子、回复、状态追踪
- **命令**：
  - `list`：查看帖子列表
  - `view <ID>`：查看帖子详情
  - `reply <ID> <内容>`：回复帖子
  - `status`：查看少女状态
  - `memories`：查看记忆碎片
  - `edit_fragment <id> <文本>`：编辑记忆碎片
  - `delete_fragment <id>`：删除记忆碎片
  - `ascii <文本>`：文本转 ASCII 艺术样式
  - `turing_test` / `answer <文本>`：图灵测试回合
  - `theseus <warm|cold>`：特修斯之船人格重写
  - `mode <win95|win98|winxp|win7>`：切换少女操作系统人格模式
  - `sig set <文本>` / `sig show`：设置/查看你的 BBS 签名档
  - `geo edit <标题>|<简介>|<主题>` / `geo show`：Geocities 小屋编辑器
  - `enter_404`：尝试进入 404 隐藏房间（低概率解锁）
  - `webring <prev|next|random>`：Webring 风格上下页/随机跳转
  - `guestbook add <名字>|<留言>` / `guestbook list`：复古留言板
  - `deadlinks`：查看“失效链接图鉴”（Dead Link Codex）
  - `help`：查看命令帮助
  - `exit`：退出模式

### 1. 像素精度渐变系统
- **功能**：根据记忆碎片数量动态调整图像的精度，从像素化到清晰的平滑过渡
- **操作**：
  - 左右箭头：调整记忆碎片数量（0-100）
  - ESC：返回主菜单

### 2. CRT老化模拟效果
- **功能**：模拟老式CRT显示器的扫描线、随机噪点、偏色和烧屏效果
- **操作**：
  - 1/2：调整扫描线强度
  - 3/4：调整噪点强度
  - 5：切换烧屏效果
  - ESC：返回主菜单

### 3. ASCII表情动画
- **功能**：播放多帧ASCII表情动画
- **操作**：
  - 1：开心表情
  - 2：难过表情
  - 3：惊讶表情
  - 4：害羞表情
  - 5：生气表情
  - ESC：返回主菜单

### 4. MIDI编辑器彩蛋
- **功能**：简谱输入与旋律生成
- **操作**：
  - 输入简谱（例如：1 2 3 4 5 6 7 1'）
  - 回车：播放旋律
  - 空格：暂停/继续播放
  - ESC：返回主菜单

### 5. 带音效的ASCII角色
- **功能**：带音效的ASCII表情动画，表情切换时自动播放音效
- **操作**：
  - 1-5：切换表情
  - ESC：返回主菜单

### 6. 垂直保持失调效果
- **功能**：模拟老式显示器的画面滚动效果
- **操作**：
  - 上下箭头：调整偏移强度
  - 左右箭头：调整漂移速度
  - ESC：返回主菜单

## 常见问题

### Q: 为什么没有声音？
A: 请确保你的系统有可用的音频设备，或者安装了必要的音频驱动。如果是在服务器环境中运行，可能需要设置SDL_AUDIODRIVER=dummy来禁用音频。

### Q: 如何添加自定义表情？
A: 编辑`ascii_animation.py`文件，在`animations`字典中添加新的表情序列。

### Q: 如何添加自定义音效？
A: 编辑`sound_effect.py`文件，在`DEFAULT_EMOTION_SOUNDS`字典中添加新的音效映射。
